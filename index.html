Example usage

Methods

  • $.responsive()

    Instantiate responsive-js.

  • $.responsive().from( from, to, key )

    Specifies a range. After specifying a range you can start adding callbacks with the $.responsive().enter() and $.responsive().exit() methods.

    Is recommended to set ranges from lowest to greatest and to avoid overlapping ranges.

    Arguments

    from (integer)

    Lowest window width in range.

    to (string)

    Hightest window width in range.

    key (string)optional

    Range identifier. This identifier will be passed as an argument on each callback.

  • $.responsive().enter( callback )

    Sets a callback function to be executed whenever the window width is between the current range

    Arguments

    callback (function)

    Callback function. Will be called with four arguments:

    from (integer)
    Lowest window width in current range.
    to (integer)
    Highest window width in current range.
    window_width (integer)
    Current window width.
    kind (string)
    Either this is an "enter" or "exit" event.
    key (string)
    String identifier of current range. If no identifier was given when calling the $.responsive().from() method, this argument will be undefined.

  • $.responsive().exit( callback )

    Sets a callback function to be executed when the window width is no longer in range.

    Arguments

    callback (function)

    Will be called with four arguments:

    from (integer)
    Lowest window width in current range.
    to (integer)
    Highest window width in current range.
    window_width (integer)
    Current window width.
    kind (string)
    Either this is an "enter" or "exit" event.
    key (integer)
    String identifier of current range. If no identifier was given when calling the $.responsive().from() method, this argument will be undefined.
